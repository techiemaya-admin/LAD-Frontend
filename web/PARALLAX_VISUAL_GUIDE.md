## Animation Flow Visualization

### SCROLL PARALLAX EFFECT

```
SCROLL STATE 0%
(At top of section)

┌─────────────┬─────────────────────┐
│ LEFT        │ CARD 1 (y: 0px)     │
│ (sticky)    │ CARD 2 (y: 0px)     │
│             │ CARD 3 (y: 0px)     │
│             │ CARD 4 (y: 0px)     │
└─────────────┴─────────────────────┘
        ↓ USER SCROLLS DOWN ↓

SCROLL STATE 25%
┌─────────────┬─────────────────────┐
│ LEFT        │ CARD 1 (y: +12px)   │  (Moving down slower)
│ (sticky)    │ CARD 2 (y: +20px)   │  (Moving down faster)
│             │ CARD 3 (y: +27px)   │  (Moving down even faster)
│             │ CARD 4 (y: +35px)   │  (Moving down fastest)
└─────────────┴─────────────────────┘
        ↓ USER SCROLLS DOWN ↓

SCROLL STATE 50%
┌─────────────┬─────────────────────┐
│ LEFT        │ CARD 1 (y: +25px)   │
│ (sticky)    │ CARD 2 (y: +40px)   │  ← Creates "wave" effect
│             │ CARD 3 (y: +55px)   │     where lower cards
│             │ CARD 4 (y: +70px)   │     lag behind upper ones
└─────────────┴─────────────────────┘
        ↓ USER SCROLLS DOWN ↓

SCROLL STATE 100%
(At bottom of section)

┌─────────────┬─────────────────────┐
│ LEFT        │ CARD 1 (y: +50px)   │  (Moved least)
│ (sticky)    │ CARD 2 (y: +80px)   │
│             │ CARD 3 (y: +110px)  │
│             │ CARD 4 (y: +140px)  │  (Moved most)
└─────────────┴─────────────────────┘

RESULT: Cards have "staggered" positions
        creating visual depth & interest
```
